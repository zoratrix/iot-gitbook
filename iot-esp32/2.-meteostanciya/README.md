## Работа с датчиком

То резистор не нужен и остается подключить только +, - и сигнал в соответствии с надписями на плате.

Напишем простую программу, суть которой - получать значения температуры и влажности от датчика и выводить их на экран компьютера (в терминал).

В первую очередь после создания нового файла нужно подключить библиотеку для работы с датчиком. Она называется "DHT sensor library". Если она установлена, то это можно сделать так:

Если в списке библиотек нужная не нашлась, значит, ее необходимо скачать. Удобнее всего это сделать прямо в Arduino IDE, нажав (в версии 2) кнопку ![](<../../.gitbook/assets/image (73).png>) в панели слева. В открывшейся панели в поле поиска сверху вводим название библиотеки, находим в списке нужную и нажимаем кнопку "Install" (она будет вместо update на скриншоте):

<figure><img src="../../.gitbook/assets/image (74).png" alt=""><figcaption></figcaption></figure>

Следующий шаг - создать датчик в программе (создать экземпляр класса датчика в своем коде). Для этого используем код:

```arduino
DHT dht_sensor(5, DHT22);
```

* `dht_sensor` - это название датчика в вашем коде, его можно изменить;&#x20;
* `5` - это номер пина, в который подключен датчик;
* `DHT22` - это тип датчика (если он белого цвета, то это DHT22, если синего - DHT11);

Наша задача - считывать показания с датчика. Их нужно куда-то записывать. Поэтому далее нужно создать переменные, в которые мы будем записывать значения температуры (temp) и влажности (hum, от английского humidity = влажность). Тип переменных – float, так как значения могут иметь дробную часть:

```arduino
float temp = 0.0;
float hum = 0.0; 
```

Теперь переходим к функции `setup()`.&#x20;

Внутри цикла setup нам нужно запустить датчик командой `dht_sensor.begin()` и передачу данных на экран компьютера уже знакомой командой `Serial.begin(115200)`:

```arduino
void setup() {
  
  dht_sensor.begin();
  Serial.begin(115200);

}
```

Внутри функции `loop()` нужно делать две вещи: считывать и записывать показания датчика в переменные `temp` и `hum`, а затем выводить их на экран.

Сначала считывание: получить показания температуры можно с помощью функции (метода) `getTemperature()`, влажности, соответственно, `getHumidity()`. В начале через точку, конечно, нужно указать название датчика: `название.getTemperature()`. Полученные значения записываем в переменные `temp` и `hum`. Тогда код получается такой:

```arduino
void loop() {
  
  temp = dht_sensor.getTemperature();
  hum = dht_sensor.getHumidity();

}
```

Теперь выводим на экран:

```arduino
Serial.print("Температура: ");
Serial.println(temp);
Serial.print("Влажность: ");
Serial.println(hum);
```

Если оставить так, то данные будут публиковаться слишком быстро. Нам бы хотелось, чтобы это происходило медленнее - например, выводить значения каждые 2 секунды. Проще всего это сделать с помощью команды `delay` - добавим ее в конце:

```arduino
void loop() {

  temp = dht_sensor.getTemperature();
  hum = dht_sensor.getHumidity();

  Serial.print("Температура: ");
  Serial.println(temp);
  Serial.print("Влажность: ");
  Serial.println(hum);
  
  delay(2000);
}
```

Код целиком:

```arduino
#include <DHT.h> // библиотеки для датчика DHT
#include <DHT_U.h>

DHT dht_sensor(5,DHT22); // инициализация датчика

float temp = 0.0; // переменная для записи температуры
float hum = 0.0; // переменная для записи влажности

void setup() {

  dht_sensor.begin(); // запуск датчика
  Serial.begin(115200); // запуск передачи данных в терминал

}

void loop() {

  temp = dht_sensor.getTemperature(); //считываем значение температуры в переменную temp
  hum = dht_sensor.getHumidity(); //считываем значение влажности в переменную hum

  Serial.print("Температура: ");
  Serial.println(temp); // выводим температуру в терминал
  Serial.print("Влажность: ");
  Serial.println(hum); //выводим влажность в терминал
  
  delay(2000);

}
```

<mark style="background-color:orange;">**НО!**</mark>

Для такой простой программы delay() достаточно, но для кода с передачей данных по MQTT delay может помешать, потому что он останавливает выполнение всего кода программы на указанное время. Подробнее тут: [функции времени](https://alexgyver.ru/lessons/time/), [многозадачность в Arduino](https://alexgyver.ru/lessons/how-to-sketch/).

Поэтому стоит попробовать сделать таймер по-другому. Мы используем библиотеку от AlexGyver под названием GyverTimer (скачайте ее при необходимости). Работать с ней просто:

* подключаем библиотеку в начале кода

```
#include <GyverTimer.h>
```

* создаем таймер там же в начале (до цикла setup()). MS - эт означит, что считаем в миллисекундах. dhtTimer - название таймера.

```arduino
GTimer dhtTimer(MS);  
```

* в цикле setup() задаем время, через которое будет срабатывать таймер (например, каждые 2000 мс)

```arduino
dhtTimer.setInterval(2000);
```

* когда таймер сработает, значение функции название\_таймера.isReady() станет равно 1 (true). В это момент мы можем выполнить какое-то действие, например, опубликовать значение температуры и влажности
* то есть, ЕСЛИ (таймер.isReady) ТО {публикуем сообщения}

```arduino
  if (dhtTimer.isReady()){ // когда таймер dhtTimer сработал (прошло 2 сек)
    Serial.print("Температура: ");
    Serial.println(temp);
    Serial.print("Влажность: ");
    Serial.println(hum);
  }
```

Готово! **Код целиком с таймером:**

```arduino
#include <GyverTimer.h> // библиотека для создания таймеров
#include <DHT.h> // библиотеки для датчика DHT
#include <DHT_U.h>

DHT dht_sensor(5,DHT22); // инициализация датчика
GTimer dhtTimer(MS);   //инициализация таймера для публикации значений темп. и влажн.

float temp = 0.0; // переменная для записи температуры
float hum = 0.0; // переменная для записи влажности

void setup() {

  dht_sensor.begin(); // запуск датчика
  Serial.begin(115200); // запуск передачи данных в терминал
  dhtTimer.setInterval(2000); // установка времени таймера

}

void loop() {

  temp = dht_sensor.getTemperature(); //считываем значение температуры в переменную temp
  hum = dht_sensor.getHumidity(); //считываем значение влажности в переменную hum

  if (dhtTimer.isReady()){ // когда таймер dhtTimer сработал (прошло 2 сек)
    Serial.print("Температура: ");
    Serial.println(temp); // выводим температуру в терминал
    Serial.print("Влажность: ");
    Serial.println(hum); //выводим влажность в терминал
  } 
}
```
