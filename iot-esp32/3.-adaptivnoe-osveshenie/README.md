# 3. Адаптивное освещение

Устройство для разработки в этом разделе - система адаптивного освещения. Суть заключается в том, что лампа горит не с постоянной яркостью, а подстраивается под уровень освещенности в помещении. Например, в полной темноте лампа горит на полной яркости (100%), если в комнате становится немного светлее, то яркость лампы понижается, и, когда в комнате светло, лампа горит на минимальной яркости или выключается совсем.

Для этого понадобится измерять уровень освещенности. Проще всего это сделать с помощью фоторезистора:

<figure><img src="../../.gitbook/assets/image (63).png" alt=""><figcaption></figcaption></figure>

Фоторезистор - полупроводниковый прибор, изменяющий величину своего сопротивления при облучении светом.

Мы будем использовать фоторезистор на модуле:

<figure><img src="../../.gitbook/assets/image (93).png" alt=""><figcaption></figcaption></figure>

На плате модуля освещенности так же распаяны: компаратор LM393 и подстроечный резистор. Это дает возможность настроить сенсор освещенности на срабатывание при любом уровне светового потока.

### Задание 1. Считать и вывести значение с датчика

Наша задача: написать программу, которая будет считывать показания освещенности с датчика (фоторезистора) и выводить их в терминал каждые 2 секунды.

**Для считывания показаний достаточно использовать функцию `analogRead(номер пина)`**, которая возвращает число в диапазоне от 0 до 1024 (напряжение от 0 до 1 вольта). Причем 1024 - это полное отсутствие света, а 0 = 100% освещенность.

* Не забудьте, что для выполнения действий с определенной периодичностью удобно использовать библиотеку GyverTimer.
* Не забудьте также создать переменную, в которую вы будете записывать яркость.
* Также полезной может оказаться функция `map()`.

Эта  функция пропорционально переносит значение (`знач`) из текущего диапазона значений (_**`старое_меньшее`**_ .. _`старое_большее`_) в новый диапазон (_`новое_меньшее` .. `новое_большее`_), заданные числами или переменными:

```arduino
map(знач, старое_меньшее, старое_большее, новое_меньшее, новое_большее);
```

Например, мы получаем значения с датчика в переменную `old` в диапазоне от 1 до 1000 единиц. Но выводить для пользователя мы хотим значение в процентах (от 1 до 100). Новое значение в процентах нужно записать в переменную new. Тогда код будет выглядеть так:

```arduino
new = map(old, 0, 1000, 0, 100);
```

Если old = 500, то new будет равно 50, если old = 670, то new = 67, и так далее. (В этом конкретном примере можно было бы просто обойтись делением на 10, но без map() не обойтись, когда нам нужно преобразовать, например, диапазон от (0...1024) в (68...337))

Также map() может **переворачивать** диапазон. Например, мы получаем значения уровня воды с датчика  от 0 до 255, при этом 255 = воды нет, а 0 = полное заполнение бака. Чтобы пользователю было удобно, стоит вывести значение в процентах, но будет сложно воспринимать информацию, если мы преобразуем в 0...100 напрямую, ведь получится, что 100% - это полностью пустой бак, а 0% - это заполненный. Поэтому новые значения меньшего и большего при использовании map() можно **поменять местами**:

```arduino
level_new = map(level_old, 0, 255, 100, 0);
```

<summary>Решение</summary>

Перед циклом setup добавляем библиотеку GyverTimer (для того, чтобы выводить данные раз в 2 секунды), указываем имя и адрес датчика, создаем переменную brightness типа int для хранения значения яркости, а также инициализируем таймер:

```arduino

#include <GyverTimer.h> // библиотека для таймера

#define lightSensor A0  // называем пин датчика освещенности
int brightness = 0;     // создаем переменную для записи значения с датчика

GTimer lightTimer(MS);  // инициализируем таймер

```

В цикле setup запускаем передачу данных и указываем время срабатывания таймера:

```arduino
void setup() {
  
  Serial.begin(115200);         // запускаем передачу данных в терминал
  lightTimer.setInterval(2000); // задаем время таймера = 2 сек

}
```

В цикле loop нам нужно&#x20;

* получить данные с датчика с помощью функции analogRead(),&#x20;
* преобразовать диапазон из непонятных 0...1024 в проценты 0...100,
* при срабатывании таймера вывести значение на экран (конечно, с подписью)

```arduino
void loop() {
  
  brightness = analogRead(lightSensor);           // считываем значение с датчика
  int percent = map(brightness, 0, 1024, 100, 0); // преобразуем диапазоны
  if (lightTimer.isReady()){                      // когда таймер сработал
    Serial.print("Освещенность:");                // вывести в терминал надпись
    Serial.println(percent);                      // вывести в терминал значение (преобразованое, в процентах)
  } 
}
```

Весь код:


### Задание 2. Управляем яркостью 1 светодиода с помощью датчика

Обычно, чтобы просто включить светодиод с постоянной яркостью, мы используем функцию `digitalWrite(название_светодиода, 1);` Однако, можно управлять тем, насколько ярко светит светодиод.&#x20;

Для этого нужно использовать функцию `analogWrite(название_светодиода, яркость)`, где яркость - это число от 0 до 255 (0 - не горит, 255 - самый яркий).

Напишите программу, которая будет менять яркость светодиода в зависимости от текущего уровня освещенности: если полностью темно - то светодиод горит максимально ярко, если полностью темно - то светодиод не горит (или очень тускло).

<details>

<summary>Решение</summary>

В первую очередь записываем и называем пины датчика и светодиода через #define, а также создаем переменную для значения с датчика:

```arduino
#define lightSensor A0    // называем пин датчика
#define lamp D1           // называем пин светодиода
int brightness = 0;       // создаем переменную для записи показаний с датчика
```

Затем в цикле setup указываем режим работы пина светодиода как выходной:

```arduino
void setup() {
  pinMode(lamp, OUTPUT);  // режим работы пина светодиода = выход (OUTPUT)
}
```

Наконец, в цикле loop сначала считываем значение с датчика с помощью analogRead(), затем преобразуем диапазон с помощью map от 0...1024 в 0...255 и затем передаем с помощью функции analogWrite полученное значение на светодиод.

```arduino
void loop() {
  brightness = analogRead(lightSensor);                     // считываем значение с датчика в переменную 

  int brightnessToLamp = map(brightness, 0, 1024, 0, 255);  // преобразуем значение с дачтика в яркость светодиода
  analogWrite(D1, brightnessToLamp);                        // передаем значение яркости на светодиод

}
```
