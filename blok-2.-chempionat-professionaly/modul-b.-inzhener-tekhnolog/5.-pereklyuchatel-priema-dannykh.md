# 5. Переключатель приема данных

По заданию также нужно реализовать **переключатель**, который будет **включать/выключать прием данных** вообще.

Сделаем это аналогично переключателю перевода значений

![](https://robotus.gitbook.io/iot-prof/~gitbook/image?url=https%3A%2F%2F2624546846-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FCmu7bhZlqFUPvAOzeuSv%252Fuploads%252FZWrgYHumYdXlVdYSlQi7%252Fimage.png%3Falt%3Dmedia%26token%3D1833856b-0a1a-4f63-91fd-1fd5320dc606\&width=768\&dpr=4\&quality=100\&sign=774a2b8a\&sv=2)

Код функции:

```javascript
global.set("poluchatSwitch", msg.payload);
return msg;
```

За получение данных у нас отвечает функция "получить все данные <название устройства>", например:

![](https://robotus.gitbook.io/iot-prof/~gitbook/image?url=https%3A%2F%2F2624546846-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FCmu7bhZlqFUPvAOzeuSv%252Fuploads%252F4YnkKVSLlkrXEZjkYerF%252Fimage.png%3Falt%3Dmedia%26token%3D0143d97d-7536-42ea-a63b-b751377991ad\&width=768\&dpr=4\&quality=100\&sign=ac93f995\&sv=2)

Изменим код этой функции.

Было так:

```javascript
if (msg.payload.name == "Robot1_2"){
    return msg
}
```

Станет так:

```javascript
if (msg.payload.name == "Robot1_2" && global.get("poluchatSwitch")){
    return msg
}
```

То есть, передавать данные (от робота 1\_1), только если название = robot1\_1 **И** переключатель приема данных включен
