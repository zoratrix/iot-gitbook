# Проект: Парктроник

* **Задача:** Ультразвуковой датчик расстояния и пьезо-пищалка (buzzer). Чем ближе препятствие, тем чаще пищит зуммер.
* **Практика:** Работа с библиотекой для датчика, функция map() для преобразования расстояния в частоту писка, функция tone().
*   **Документация:**

    1. **Объектная модель (упрощенно):** Вводим понятие объекта. "У нас есть объект Парктроник. У него есть свойство расстояние и метод издать\_звук()".
    2. **Таблица параметров мониторинга:**

    | Параметр   | Ключ     | Тип данных | Диапазон   | Критическое значение |
    | ---------- | -------- | ---------- | ---------- | -------------------- |
    | Расстояние | distance | Integer    | 0 - 400 см | < 10 см              |

3. **Алгоритм реакции на критические значения:** Как на рис. 3.1 в вашем файле. "Если `distance` < 10, то пищать непрерывно".
