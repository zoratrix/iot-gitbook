# Проект: Система контроля доступа по RFID

* **Суть:** Модернизация "Панели доступа" с использованием RFID-меток.
* **Оборудование:** Arduino Nano, считыватель RFID RC-522, карта и метка, дисплей 1602 I2C, сервопривод MG90S, реле.
* **Задачи:**
  1. Запрограммировать систему на распознавание "своей" карты/метки.
  2. При поднесении верной карты — открыть "замок" (повернуть сервопривод на 90 градусов или щелкнуть реле).
  3. При поднесении чужой карты — выдать сигнал тревоги.
  4. Добавить режим "программирования", когда можно добавить новую карту в память (EEPROM).
* **Чему учит:** SPI-интерфейс, работа с библиотекой MFRC522, работа с энергонезависимой памятью (EEPROM).
* **Документация:** Объектная модель ("Считыватель", "Замок", "Карта доступа"). Таблица параметров оборудования.
