# Проект: Интерактивный ночник

### Задание

Собрать схему и написать программу, где каждое нажатие кнопки переключает цвет светодиода по циклу: <mark style="background-color:red;">Цвет1</mark> -> <mark style="background-color:green;">Цвет2</mark> -> <mark style="background-color:blue;">Цвет3</mark> -> Выкл -> <mark style="background-color:red;">Цвет1</mark>...

Также:

* Разработайте **блок-схему** [blok-skhemy.md](../../dokumentaciya/blok-skhemy.md "mention") алгоритма работы программы
* Разработайте **тест-кейсы** [test-keisy.md](../../dokumentaciya/test-keisy.md "mention") для проверки работоспособности

### Подсказки по выполнению

**Схема устройства:**

* Подключаем кнопку, как разобрали ранее (пин 2, стягивающий резистор на GND).
* Подключаем RGB-светодиод (общий катод -> GND; выводы R, G, B -> через резисторы к пинам 9, 10, 11)

**Алгоритм программы:**

* Первый раз нажали кнопку -> включить цвет1
* Второй раз нажали кнопку -> включить цвет2
* Третий раз нажали кнопку -> включить цвет3
* Четвертый раз нажали кнопку -> выключить лампу

Подумайте, как изобразить этот алгоритм в виде блок-схемы? Как запоминать количество нажатий на кнопку?

<summary>Подсказка по коду</summary>

Начало кода - все объявляем, считываем значение кнопки

```arduino
#include <RGBLED.h>
RGBLED rgb(11, 10, 9); 
#define button 2
int button_state = 0; // Текущее состояние кнопки

void setup() {
  pinMode(button, INPUT);
}
void loop() {
  button_state = digitalRead(button) // считываем сигнал с кнопки

```

Далее нам нужно запоминать, сколько раз уже была нажата кнопка, и в зависимости от этого числа включать разный цвет.

Мы можем создать переменную «счетчик», в которую будем записывать, какое сейчас нажатие. После каждого нажатия будем увеличивать счетчик на +1

Разберемся со счетчиком. Нам нужно зафиксировать не просто что «кнопка нажата» (могут быть ложные срабатывания), а момент нажатия кнопки, то есть изменение ее состояния с 0 на 1.

Добавим переменную `int old_button_state` (изначально = 0) перед функцией `setup`. В коде `loop` тогда можем написать:

<pre class="language-arduino"><code class="lang-arduino">if (button_state == 1 &#x26;&#x26; old_button_state == 0){
<strong>    //… здесь что-то произойдет в момент нажатия кнопки
</strong>}
</code></pre>

Итак, в момент нажатия кнопки мы увеличиваем счетчик на +1 (то есть к старому значению счетчика прибавляем 1. а значение счетчика лежит в переменной counter, то есть мы прибавляем +1 и записываем в ту же переменную):

```arduino
if (button_state == 1 && old_button_state == 0){
    counter = counter + 1;
}
```

А чему равно old\_button\_state? В самом начале кода мы  создали эту переменную и написали =0

Нам нужно менять ее значение - обновлять ее. Будем делать это после нажатия:

```arduino
if (button_state == 1 && old_button_state == 0){
    counter = counter + 1;
}
old_button_state = button_state;
```

После этого мы можем вывести цвет в зависимости от числа на счетчике:

```arduino
if (counter == 0) {
    // Все выключено
    rgb.setRGB(0,0,0);
} else if (counter == 1) {
    // Красный
    rgb.setRGB(255,0,0);
} else if (counter == 2) {
    // Зеленый
    rgb.setRGB(0,255,0);
} else if (counter == 3) {
    // Синий
    rgb.setRGB(0,0,255);
}
```

Код целиком:

```arduino
void loop() {
  button_state = digitalRead(button); //считываем сигнял с кнопки
  if (button_state == 1 && old_button_state == 0){ // если произошел клик на кнопку
    counter = counter + 1; // счетчик +1
    delay(50); // маленькая задержка для стабильности
    if (counter > 3) { //если счетчик досчитал до 3
      counter = 0; //сбрасываем его до 0, чтобы начать с начала
    }
  }

  old_button_state = button_state; //обновляем переменную со старым состоянием кнопки

  if (counter == 0) {
    rgb.setRGB(0,0,0); // Все выключено
  } else if (counter == 1) {
    rgb.setRGB(255,0,0); // Красный
  } else if (counter == 2) {
    rgb.setRGB(0,255,0); // Зеленый
  } else if (counter == 3) {
    rgb.setRGB(0,0,255); // Синий
  }
}
```
