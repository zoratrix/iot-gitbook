# Проект: Кодовый замок

## Описание

Вы разрабатываете кодовый замок для секретного чемодана. Перед пользователем находится клавиатура 4х4, светодиод или зуммер (пищалка) и дисплей. Для того, чтобы открыть замок, пользователь вводит пароль известной длины. Если пароль верный - доступ разрешен. Если нет - доступ запрещен.

Разработайте алгоритм (блок-схему), тест-кейсы, электрическую схему, код программы.

### Детали проекта:

* Длина пароля должна быть фиксированной (4 символа)
* При правильном вводе пароля система должна:
  * Выводить сообщение "SUCCESS"
  * Активировать светодиодный индикатор или зуммер
  * Поддерживать состояние "открыто" в течение 10 секунд
* При неправильном вводе пароля система должна:
  * Выводить сообщение "ACCESS DENIED"
  * Блокировать ввод на 5 секунд
* Дополнительно: при вводе каждого символа на дисплее должен отображаться символ '\*' (маскирование ввода)

### Структура программы:

```
1. ИНИЦИАЛИЗАЦИЯ СИСТЕМЫ (функция setup)
   ├─ Инициализация дисплея
   ├─ Настройка пинов ввода/вывода
   └─ Вывод приветственного сообщения

2. ОСНОВНОЙ ЦИКЛ
   ├─ Ожидание ввода символа с клавиатуры
   ├─ Отображение '*' при каждом вводе
   ├─ Накопление введенных символов в буфер
   └─ При достижении 4 символов → ПРОВЕРКА (отдельная функция)

3. ФУНКЦИЯ ПРОВЕРКИ ПАРОЛЯ
   ├─ Сравнение с эталонным паролем
   ├─ Если верно → СЦЕНАРИЙ УСПЕХА
   └─ Если неверно → СЦЕНАРИЙ ОШИБКИ

4. СЦЕНАРИЙ УСПЕХА
   ├─ Вывод "SUCCESS"
   ├─ Включение светодиода
   ├─ Задержка 10 секунд
   └─ Возврат к шагу 1

5. СЦЕНАРИЙ ОШИБКИ
   ├─ Вывод "ACCESS DENIED"
   ├─ Задержка 5 секунд
   └─ Возврат к шагу 1
```

Теперь нужно написать код функции проверки пароля checkPassword().&#x20;

Сначала создадим функцию и внутри нее добавим переменную password\_correct типа bool (значения true или false), чтобы в нее записывать верный пароль (true) или неверный (false):

Теперь напишем код проверки. Нам нужно проверить каждый символ во введенном пользователем пароле (а он хранится в массиве input). Сделаем это с помощью цикла for: i будет принимать значения от 0 до password\_dlina (например, 4), и будем сравнивать элемент из массива input (введенный пользователем) под номером i и элемент из массива password (предустановленный пароль) под номером i. Если все элементы совпадут, то пароль верный. Если какой-то элемент не совпал, то выходим из цикла командой break:

```arduino
void checkPassword() {
  // создаем переменную для записи верный/неверный пароль
  bool password_correct = true; //пусть изначально эта переменная true, если после проверки она осталась true, то пароль верный
  
  // Циклом проверяем каждый символ
  for (int i = 0; i < password_dlina; i++) {
    if (input[i] != password[i]) {
      password_correct = false; // Если хоть один символ не совпал, пароль неверный
      break; // Прерываем цикл
    }
  }
 // Очищаем введенный пароль для следующей попытки
  input_counter = 0;
}
```

  // создаем переменную для записи верный/неверный пароль
  bool password_correct = true;
  
  // Циклом проверяем каждый символ
  for (int i = 0; i < password_dlina; i++) {
    if (input[i] != password[i]) {
      password_correct = false; // Если хоть один символ не совпал, пароль неверный
      break; // Прерываем цикл
    }
  }

  // Очищаем введенный пароль для следующей попытки
  input_counter = 0;

  if (password_correct) {
    // Действия при правильном пароле
    lcd.clear();
    lcd.setCursor(0, 0);
    lcd.print("*** SUCCESS ***");
    digitalWrite(lampa, HIGH); // Зажигаем светодиод

    // "Закрываем замок"
