# Светодиодная лента

## Описание

Светодиодная лента - это цепочка светодиодов (обычных или RGB). Светодиодная лента может быть:

* обычная - все светодиоды управляются как один, горят одновременнно, с одинаковой яркостью и т.п.
* адресная - каждым светодиодом можно управлять отдельно

Мы будем говорить об адресной светодиодной ленте. Если вас интересует тема таких лент, посмотрите подробнее [тут](https://alexgyver.ru/ws2812_guide/).&#x20;

## Подключение

Команды в ленте передаются от диода к диоду, паровозиком. У ленты есть начало и конец, направление движение команд на некоторых моделях указано стрелочками. Даже если стрелок нет, на любом отрезке ленты с двух концов есть обозначение - вход (IN) и выход (OUT). Соотвественно, к arduino для управления подключим IN, а если захотим удлинить ленту - соединим OUT первой с IN второй. Также подключаем GND и 5V в arduino.

## Программирование

Самой функциональной и известной библиотекой является [FastLED](https://github.com/FastLED/FastLED), можно установить через диспетчер библиотек по названию FastLED. В библиотеке ОЧЕНЬ много возможностей, см. официальную [вики-документацию](https://github.com/FastLED/FastLED/wiki/Basic-usage), а также [полный список](https://fastled.io/docs/) модулей, классов и функций. Мы рассмотрим основные.

Для начала объявляем через #define количество светодиодов и пин подключения ленты:

```arduino
#define lenta_pin 3     // пин
#define lenta_num 8    // количество светодиодов
```

Подключаем библиотеку:

```cpp
#include <FastLED.h>
```

Создаём массив цветов, он имеет тип данных `CRGB`. Это специальный тип данных, из библиотеки.  Размер – сколько светодиодов в ленте, то есть наша константа:

```cpp
CRGB leds[lenta_num];
```

В блоке `setup` нам нужно передать в библиотеку информацию о подключенной ленте, а также подключить созданный выше массив. Чтобы программа знала, откуда брать информацию о цветах. По порядку указывается тип ленты, пин, порядок цветов. В круглых скобках – имя нашего массива, а также его размер – длину ленты:

```cpp
void setup() {  
  FastLED.addLeds<WS2812, lenta_pin, GRB>(leds, lenta_num);
}
```

Отлично! Всё настроено и готово к работе.

Теперь мы можем поуправлять светодиодами в ленте. Напомню, что светодиоды в ленте в нашей программе - это массив из знчений формата CRGB. То есть, чтобы зажечь 1 светодиод нам нужно написать leds\[номер\_светодиода] = цвет; Цвет CRGB можно задать по-разному:

```arduino
leds[0] = 0xFF44DD; //задать цвет 16-ричным кодом (hex)
leds[0] = CRGB::HotPink; //задать цвет по названию (вместо HotPink - название цвета)

//для 1 светодиода задать отдельно 3 канала цвета:
leds[0].r = 255; 
leds[0].g = 68; 
leds[0].b = 221;

//или такие опции:
leds[0].setRGB(255, 68, 221); // цвет в формате RGB
leds[0].setHSV(224, 187, 255);  // цвет в формате HSV
leds[0].setHue(224); // радуга от 0 до 255
```

Если вы задаете цвет как CRGB::HotPink названия цветов можно посмотреть [тут](https://github.com/FastLED/FastLED/wiki/Pixel-reference#colors).

Эти команды только записывают цвет, но не показывают его на ленте. Чтобы обновить цвет ленты, нужно вызвать команду:

```arduino
FastLED.show();
```

Это мы задаем цвет одного светодиода. Получается, если хотим покрасить всю ленту, простейший вариант:

```arduino
leds[0] = CRGB::HotPink;
leds[1] = CRGB::HotPink;
leds[2] = CRGB::HotPink;
leds[3] = CRGB::HotPink;
leds[4] = CRGB::HotPink;
leds[5] = CRGB::HotPink;
leds[6] = CRGB::HotPink;
leds[7] = CRGB::HotPink;
FastLED.show();
```

Конечно, это не оптимальный способ, особенно, если лента будет длиннее. Можем использовать цикл for:

```arduino
for (int i = 0; i <8; i++){
    leds[i] = CRGB::HotPink;
}
FastLED.show();
```

Так мы покрасим в розовый все светодиоды от i = 0 до i = 7.

Для заливки всей ленты одним цветом так же есть отдельная команда:

```arduino
FastLED.showColor(CRGB::HotPink);
```

Она сразу отображает цвет (FastLED.show(); не нужна)!

Также есть команда для выключения всей ленты:

```arduino
FastLED.clear();
```

Но заливать ленту одним цветом, конечно, скучно. Можно, например, сделать эффект радуги, растянутой на всю ленту. Для этого воспользуемся командой leds.setHue(). Она передает на ленту цвет в формате HSV

Если мы представляем цвет в HSV, то для светодиодной ленты мы вполне можем выкрутить S и V на максимум (максимальная цветность и максимальная яркость), а именно цвет менять параметром H (Hue) от красного к желтому, зеленому и далее радугой снова до красного):

<figure><img src="../../.gitbook/assets/image (36).png" alt="" width="375"><figcaption></figcaption></figure>

В библиотеке есть, как я написала ранее, команда setHue(), она делает именно это, оставляет S =255, V = 255, и меняет только значение H от 0 до 255. У нас 8 светодиодов, если мы хотим вывести радугу, поделим 255/8 и получим \~31. Тогда на первый светодиод передадим 31, на второй 31\*2, на третий 31\*3 и так далее:

```arduino
leds[0].setHue(0);
leds[1].setHue(31);
leds[2].setHue(31*2);
leds[3].setHue(31*3);
leds[4].setHue(31*4);
leds[5].setHue(31*5);
leds[6].setHue(31*6);
leds[7].setHue(31*7);
FastLED.show();
```

Конечно, то же самое можно сделать в виде цикла for:

```arduino
for(int i = 0; i<8; i++){
    leds[1].setHue(i * 31);
}
FastLED.show();
```

Или, для любого числа светодиодов (для 8 мы получили число 255/8 = 31, тогда для любого числа будет 255/num\_leds):

```arduino
for(int i = 0; i<8; i++){
    leds[1].setHue(i * 255 / lenta_num);
}
FastLED.show();
```

## Эффекты для ленты

Вот еще несколько эффектов, которые можно отобразить на ленте:

### Динамическая радуга

На основе того же кода для радуги, что мы написали ранее, можно сделать полноценную анимацию. Для этого нам нужно, чтобы цвета светодиодов менялись по кругу, например, в первый момент времени:

<figure><img src="../../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>

Затем второй шаг:

<figure><img src="../../.gitbook/assets/image (38).png" alt=""><figcaption></figcaption></figure>

Третий шаг и так далее:

<figure><img src="../../.gitbook/assets/image (39).png" alt=""><figcaption></figcaption></figure>

Это можно сделать, реализовав счетчик (в начале кода создайте переменную counter типа byte, изначально = 0)

В примере со статической радугой мы выводили все цвета так:

```arduino
leds[0].setHue(0);
leds[1].setHue(31); 
leds[2].setHue(31*2); 
leds[3].setHue(31*3); 
leds[4].setHue(31*4); 
leds[5].setHue(31*5); 
leds[6].setHue(31*6); 
leds[7].setHue(31*7); 
FastLED.show();
```

Чтобы сделать перетекание из цвета в цвет, как на картике выше, можем подвинуть все цвета на 31:

```arduino
leds[0].setHue(0+31);
leds[1].setHue(31+31); 
leds[2].setHue(31*2+31);
leds[3].setHue(31*3+31);
leds[4].setHue(31*4+31);
leds[5].setHue(31*5+31);
leds[6].setHue(31*6+31);
leds[7].setHue(31*7+31); //значение больше 255 -> вычтется 255 и отсчет начнется снова с 0
FastLED.show();
```

затем еще сдвиг на 1 цвет (еще +31):

```arduino
leds[0].setHue(0+31+31);
leds[1].setHue(31+31+31);
leds[2].setHue(31*2+31+31);
leds[3].setHue(31*3+31+31);
leds[4].setHue(31*4+31+31);
leds[5].setHue(31*5+31+31);
leds[6].setHue(31*6+31+31);
leds[7].setHue(31*7+31+31); // если значение больше 255 -> вычтется 255 и отсчет начнется снова с 0
FastLED.show();
```

И так далее. Но таким образом радуга будет слишком резкая. Если мы хотим сделать плавное перетекание цветов, можем сдвигать их не на 31, а на 1:

```arduino
leds[0].setHue(0+1);
leds[1].setHue(31+1);
leds[2].setHue(31*2+1);
leds[3].setHue(31*3+1);
leds[4].setHue(31*4+1);
leds[5].setHue(31*5+1);
leds[6].setHue(31*6+1);
leds[7].setHue(31*7+1);
FastLED.show();
```

Затем еще на 1:

```arduino
leds[0].setHue(0+1+1);
leds[1].setHue(31+1+1);
leds[2].setHue(31*2+1+1);
leds[3].setHue(31*3+1+1);
leds[4].setHue(31*4+1+1);
leds[5].setHue(31*5+1+1);
leds[6].setHue(31*6+1+1);
leds[7].setHue(31*7+1+1);
FastLED.show();
```

А в виде цикла вот так:

```arduino
for (int i = 0; i < lenta_num; i++) {
    leds[i].setHue(counter + i * 255 / lenta_num);
}
counter++;        // counter меняется от 0 до 255, это и есть сдвиг - +1, +2, +3...
FastLED.show();
delay(30);        // задержка определяет скорость движения радуги
```

### Плавное затухание

Чтобы сделать эффект плавного затухания, пригодится команда `leds[0].nscale8(скорость_затухания)`:

```arduino
for (int i = 0; i < lenta_num; i++) {
    leds[i].nscale8(200); // 200 - скорость затухания (чем меньше, тем быстрее). 255 - не гаснет.
}
```

Этот код удобно использовать как функцию в том числе с другими эффектами (вместо `FastLED.clear()`):

```arduino
void fadeAll() {
  for (int i = 0; i < lenta_num; i++) {
    leds[i].nscale8(200); // плавно гасим светодиод под номером i
  }
}
```

### Бегущий огонь

Бегкщий огонь - это поочередное вкл/выкл светодиодов от 0 до 8, включился первый, затем он выключился и включился второй сразу, затем выключился второй и включился третий и так по кругу:

🔵⚫⚫⚫⚫⚫⚫⚫

⚫🔵⚫⚫⚫⚫⚫⚫

⚫⚫🔵⚫⚫⚫⚫⚫

⚫⚫⚫🔵⚫⚫⚫⚫

```arduino
void loop() {
  // повторяем анимацию бегущего огня
  for(int i = 0; i< lenta_num; i++) {
    FastLED.clear();      // Гасим все светодиоды
    leds[i] = CRGB::Blue; // Зажигаем светодиод под номером i
    FastLED.show();       // Показываем на ленте
    delay(100);           // Ждем - это скорость движения
  }
  // Когда дошли сюда, цикл закончился и начнется снова с позиции i=0
}
```

### Случайные искры

Симпатично выглядит эффект с рандомно выспыхивающими светодиодами. Для этого можно использовать функцию `random(от X, до Y)` -  она выдает случайное число от X до Y

```arduino
void loop() {
  fadeAll() // Используем функцию fadeAll() из примера с затуханием или FastLED.clear();
  // с fadeAll выглядит красивее
  // С случайной вероятностью зажигаем новый светодиод
  if (random(0, 100) < 30) {         // Вероятность 30% появления "искры"
    int pos = random(0, lenta_num);  // Случайная позиция
    leds[pos] = CRGB::White;         // Вспыхиваем белым
  }
  FastLED.show();
  delay(30);
}
```

⚫⚫⚫🔴🔵⚫⚫⚫

⚫⚫🔴⚫⚫🔵⚫⚫

⚫🔴⚫⚫⚫⚫🔵⚫

🔴⚫⚫⚫⚫⚫⚫🔵

```arduino
void loop() {
  for(int i = 0; i < lenta_num; i++) {
    FastLED.clear();            // Выключаем все светодиоды
    leds[i] = CRGB::Green;      // Зеленый бегунок (слева направо)
    leds[7 - i] = CRGB::Purple; // Фиолетовый бегунок (справа налево)
    FastLED.show();             // Показываем на ленте
    delay(100);                 // Ждем
  }
  // Цикл закончился - начинаем заново!
}
```
