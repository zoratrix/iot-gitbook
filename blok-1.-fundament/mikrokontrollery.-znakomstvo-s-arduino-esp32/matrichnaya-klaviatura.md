# Матричная клавиатура

### Описание

**Матричная клавиатура** позволяет получить множество кнопок для ввода данных в программу, используя **меньшее количество пинов** микроконтроллера, чем как если бы это были обычные кнопки. Клавиатура с цифрами может использоваться для создания электронных замков и сейфов, охранных систем, калькуляторов и математических игр.

### Массивы в программировании

Для начала в коде нам нужно создать матрицу символов нашей клавиатуры. Это мы сделаем с помощью **массива**. Что такое массивы подробно можно почитать [тут](https://alexgyver.ru/lessons/arrays/).&#x20;

Если кратко, **массив** - это **набор однообразных переменны**х, используется, когда нам по какой-либо причине нужно хранить **не одно значение, а несколько отдельных, но в одном месте**. Например, в массиве можно хранить номера квартир в одном подъезде, имена учеников одного класса и так далее.&#x20;

Создание простейшего пустого массива выглядит так:

```
тип_данных название[размер]
```

Или с данными:

```
тип_данных название[размер] = {данные...}
```

Например, вот массив номеров квартир в 9 подъезде (пусть в этом воображаемом подъезде находятся 8 квартир с номерами с 111 по 119) (тип данных `int` - целые числа):

```cpp
int podjezd_9[8] = {11,12,13,14,15,16,17,18,19}
```

или вот массив учеников маленького 7 класса в сельской школе (обратите внимание на тип данных `String` - это тип данных для текста, он записывается в кавычках):

```cpp
String class[10] = {"Маша", "Игорь","Игнат","Вова","Леша","Вика","Юля","Маша","Марина","Миша"}
```

Такой массив - как таблица с 1 строкой и заданным количеством столбцов (это количество мы указываем в квадратных скобках). Причем у каждого ученика есть номер (например, они расположены в порядке успеваемости или роста)

<figure><img src="../../.gitbook/assets/image (142).png" alt=""><figcaption></figcaption></figure>

Получить элемент из такого массива мы можем, указав название массива и номер элемента:

```cpp
class[2] // "Игнат"
class[5] // "Вика"
class[14] // нет такого
```

Но так же бывают и многомерные массивы, например, двухмерные. Скажем, мы хотим записать в массив всех учеников класса, но по рядам, так, как они сидят в классе:

<figure><img src="../../.gitbook/assets/image (144).png" alt=""><figcaption></figcaption></figure>

В виде массива это будет выглядеть так - как массив, состоящий из массивов:

```cpp
//при объявлении массива первое число - количество строк, второе - столбуов
String class[3][4] = 
    {"Маша", "Игнат","Игорь","Миша"},
    {"Леша", "Вова","Юля","0"},
    {"Вика", "Маша","Марина","0"}
}
```

Тогда адрес Игоря будет таким:

```cpp
class[0][2] // "Игорь"
```

Case switch

Еще одна полезная конструкция, которая пригодится нам в работе с клавиатурой, но, конечно, может использоваться не только для нее.&#x20;

Для создания ветвления в программе можно использовать конструкции на `else if`, но если нужно создать ветвление по значению переменной, которая принимает известный набор значений - лучше использовать оператор выбора - `switch`.

В общем виде он выглядит так:

```arduino
switch (значение) {
    case вариант1:
        // код
        break;

    case вариант2:
        // код
        break;

    case вариант3:
        // код
        break;

    default:
        // код
        break;
}
```

Оператор перейдёт к коду под меткой с `вариантом`, который совпадает со `значением`. Если такого варианта нет - выполнится код под меткой `default`. Особенности и требования:

* `switch` принимает только **целочисленные значения**
* `case` может быть только **целочисленной константой**, известной на этапе компиляции
* `case` **не должны повторяться**
* `default` _(по умолчанию)_ будет выполнен, если не найдётся нужный вариант. **Необязательная метка**

### Программирование клавиатуры

Для работы с клавиатурой будем использовать библиотеку [SimpleKeypad](https://github.com/maximebohrer/SimpleKeypad). Для начала через #define обяъвим количество стобцов и строк

```c
#include <SimpleKeypad.h>

#define rows 4     // количество строк
#define columns 4  // количество столбцов
```

Далее нужно указать пины подключения, это нужно сделать в массивах:

```c
// пины подключения (по порядку штекера)
byte colPins[columns] = {7, 6, 5, 4};
byte rowPins[rows] = {11, 10, 9, 8};
```

Теперь нужно создать 2-мерный массив с названиями кнопок (в этом есть смысл, если на кнопках уже есть какие-то надписи, тогда можно сделать так, что надпись в коде будет соответствовать реальной кнопке. В случае с клавиатурой без надписей, обозвать кнопки можно как угодно)

```c
char klava_names[rows][columns] = {        // массив имен кнопок, можете написать свои
  {'1', '2', '3', 'A'},
  {'4', '5', '6', 'B'},
  {'7', '8', '9', 'C'},
  {'*', '0', '#', 'D'}
};
```

Наконец, из массивов пинов и массива имен кнопок создаем клавиатуру:

```c
SimpleKeypad klava((char*)klava_names, rowPins, colPins, KP_ROWS, KP_COLS);
```

Далее в функции setup запустим передачу по серийному порту:

```arduino
void setup() {
  Serial.begin(115200);
}
```

Теперь в функции loop напишем так - получаем название текущей нажатой клавиши. Если какая-то кнопка была нажата, выведем ее название в серийный порт:

```arduino
void loop() {
  // получить текущую нажатую клавишу
  // вернёт 0 символ если ничего не нажато
  char key = pad.getKey();

  // если кнопка нажата (не 0 символ)
  if (key) {
    // выводим в порт
    Serial.println(key);

    //далее в зависимости от нажатой кнопки можем делать какие-то действия
    // switch (key) {
    // case '1':
    // ...
  }
}
```
