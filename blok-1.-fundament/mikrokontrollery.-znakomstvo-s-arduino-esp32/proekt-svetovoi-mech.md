# Проект: Световой меч

Создаем самый крутой световой меч с пультом управления.&#x20;

**Оборудование:** светодиодная лента, потенциометр, кнопки

**Суть программы:** по кнопкам (не менее 3 штук) можно включить разные эффекты на ленте, например:

* кнопка 1 - бегущий огонь
* кнопка 2 - цветной градиент
* кнопка 3 - рандомные искры
* кнопка 4 - радуга (цвет не регулируется)
* кнопка 5 - выкл. все

Потенциометр используется для выбора цвета (крутим потенциометр - меняется цвет)

**Документация:** блок-схема основного алгоритма, тест-кейсы, таблица параметров (входных и выходных)

В `current_effect` будем записывать число в зависимости от выбранного по кнопке режима.  В current\_color - значение тона Hue от 0 до 255 для выбора цвета.

### Смена режимов

Как реализовать выбор режима? Просто по клику на кнопки будем менять значение current\_effect:

```arduino
// Обработка нажатий кнопок
  if (btn1.click()) current_effect = 1;
  if (btn2.click()) current_effect = 2;
  if (btn3.click()) current_effect = 3;
  if (btn4.click()) current_effect = 4;
  if (btn5.click()) current_effect = 0;
```

Ну и выбираем эффект в зависимости от значения current\_effect:

```arduino
if (current_effect == 1){
    эффект1;
} else if (current_effect == 2){
    эффект2;
} else if (current_effect == 3){
    эффект3;
} else if (current_effect == 4){
    эффект4;
} else {
    выключить_все;
}
```

или с помощью switch:

```arduino
 // Выполнение текущего эффекта
  switch (current_effect) {
    case 0: 
      выключить_все;
      break;
    case 1: 
      эффект1; 
      break;
    case 2: 
      эффект2; 
      break;
    case 3: 
      эффект3; 
      break;
    case 4: 
      эффект4; 
      break;
  }
```

### Выбор цвета

В `current_color` будем записывать значение Hue выбранное с помощью потенциометра. Как это сделать (в loop, конечно)?&#x20;

С потенциометра через пин A0 (или другой пин A) мы получаем значение от 0 до 1023. Нам нужно использовать это значение как цвет Hue, но мы же знаем, что значение Hue меняется от 0 до 255. То есть нам надо преобразовать значение из диапазона потенциометра 0...1023 в диапазон 0...255. Это можно сделать вручную с помощью математического выражения, а можно использовать функцию map.

Команда строится так:  `map(значение, старое min, старое max, новое min, новое_max)`

В нашем случае: `map(значение потенциометра, 0, 1024, 0, 255)` &#x20;

```arduino
current_color = map(analogRead(potentometer_pin), 0, 1023, 0, 255);
```
