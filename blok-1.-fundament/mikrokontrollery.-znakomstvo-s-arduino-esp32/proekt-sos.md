# Проект: SOS

### Задание

Запрограммировать светодиод, который циклически "проигрывает" сигнал SOS азбукой Морзе (три точки, три тире, три точки).

Пусть тогда: точка = лампочка горит 0.25 секунд (250 мс), а тире = 0.75 секунд (750 мс). Алгоритм можно уточнить:

### Программа

Программу начнем так же, как и для одного светодиода. Сначала объявим, что и куда мы подключили:

```arduino
#define lampa 3           // лампа подключена в пин 3

void setup() {
  
}

void loop() {
              
}
```

затем в функции setup() указываем режим работы пина лампы:

```arduino
#define lampa 3           // лампа подключена в пин 3

void setup() {
  pinMode(lampa, OUTPUT); // лампа - выходной сигнал
}

void loop() {
              
}
```

Дальше в функции loop нам нужно написать алгоритм включения\выключения лампочки, согласно составленной ранее блок схеме. В уелом, мы можем написать все подряд:

```arduino
#define lampa 3           // лампа подключена в пин 3

void setup() {
  pinMode(lampa, OUTPUT); // лампа - выходной сигнал
}

void loop() {
  digitalWrite(lampa, 1);  //точка первая
  delay(250);
  digitalWrite(lampa, 0);
  delay(250);
  
  digitalWrite(lampa, 1); //точка вторая
  delay(250);
  digitalWrite(lampa, 0);
  delay(250);       
  
  //... и так далее...
}
```

Но на последней блок-схеме мы объединили несколько "морганий" в **группы**: `"лампа показывает S"` и `"лампа показывает O"` . Этим можно воспользоваться и в коде.&#x20;

Такая группа действий (блок кода), имеющая имя, называется _**функция**_. Функции нужны для удобства организации программы. Функцию можно **вызвать** по ее имени, чтобы выполнить содержащийся в ней код.

Вот так функция создается (вместо func - название функции):

```arduino
void func() {
    действие1;
    действия2;
}
```

Вот так она вызывается (через `имя_функции()`):

```arduino
func();
func();
```

В том месте, где будет написано func();, выполнятся действие1 и действие2

Мы, следуя блок-схеме, разработанной ранее, можем сделать две функции: "вывести букву S" и "вывести букву O". Код, в котором создается функция, нужно размещать вне функций setup() и loop(), например, до или после.

Код функции bukva\_s():

```arduino
void bukva_s() {
  digitalWrite(lampa, 1);  //точка первая
  delay(250);
  digitalWrite(lampa, 0);
  delay(250);
  
  digitalWrite(lampa, 1); //точка вторая
  delay(250);
  digitalWrite(lampa, 0);
  delay(250);       
  digitalWrite(lampa, 1); //точка вторая
  delay(250);
  digitalWrite(lampa, 0);
  delay(250); 
}
```

код функции bukva\_o():

```arduino
void bukva_o() {
  digitalWrite(lampa, 1);  //тире первое
  delay(750);
  digitalWrite(lampa, 0);
  delay(750);
  
  digitalWrite(lampa, 1); //тире второе
  delay(750);
  digitalWrite(lampa, 0);
  delay(750);       
  digitalWrite(lampa, 1); //тире третье
  delay(750);
  digitalWrite(lampa, 0);
  delay(750); 
}

```

В основной функции loop остается только вызвать созданные нами функции bukva\_s() и bukva\_o() в нужном порядке и в конце добавить delay(), чтобы после каждого слова SOS у нас была заметная пауза:

```arduino
void loop() {
  bukva_s();
  bukva_o();
  bukva_s();
  delay(3000);
}
```
