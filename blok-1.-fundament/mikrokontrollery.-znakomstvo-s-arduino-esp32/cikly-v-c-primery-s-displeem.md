# Циклы в C (примеры с дисплеем)

### Описание

**Цикл** — это управляющая конструкция, которая заставляет какой-то блок кода выполняться **несколько раз.** Циклы могут выглядеть по-разному

<figure><img src="../../.gitbook/assets/image (134).png" alt=""><figcaption></figcaption></figure>

Блок кода, который повторяется = **тело** цикла.

Перед (иногда после) циклом всегда есть **условие**, при котором он будет выполняться (даже если это условие – повторять бесконечно).

Подробно про циклы в языке C можно почитать [тут](https://www.c-cpp.ru/books/cikly) и [тут](https://alexgyver.ru/lessons-category/cpp-flow/)

### Цикл for

**Цикл `for`** – это цикл со счетчиком. Цикл со счётчиком — это цикл, в котором заранее определено **количество повторений**.

**Счетчиком** называют _переменную_, которая что-то **считает** во время работы цикла. Например, счетчик может подсчитывать, сколько раз повторилось действие. Счетчик также называют **итератором**, а одно повторение - **итерацией.**

Структура цикла for всегда вот такая:

```arduino
for (инициализация; условие; действие с счетчиком) {
    // тело цикла
}
```

Важно! Цикл for должен состоять из трёх блоков в круглых скобках. Даже если какой-то из блоков отсутствует - точка с запятой должна быть на своём месте:

```arduino
for (; условие;) {
    // тело цикла
}

for (;;) {
    // тело цикла
}
```

Идея цикла со счетчиком заключается в том, что на каждый «счет» выполняется указанное действие.&#x20;

Представьте себе, что спортсмен отжимается на «счет»: 1, 2, 3, 4, 5.

<figure><img src="../../.gitbook/assets/image (135).png" alt="" width="375"><figcaption></figcaption></figure>

В данном случае счет:

* начинается с 1 — это начальное значение счетчика: `int i = 1`
* выполняется до 5 включительно — это условие работы цикла: `i < 6;` (считаем 1, 2, 3, 4, 5; число 5 строго меньше 6)&#x20;
* после каждого отжимания значение увеличивается на 1 — это изменение счетчика: `i++`(то же самое, что `i = i+1`)

В коде это выглядело бы так:

```arduino
for (int i = 1; i < 6; i++) {
    // здесь i принимает значения 1, 2, 3, 4, 5
    отжаться();
}
```

А вот пример для Ардуино. Например, мы хотим, чтобы лампочка мигнула 4 раза:

```arduino
for (int i = 0; i < 4; i++) {
    digitalWrite(lampa, 1);
    delay(100);
    digitalWrite(lampa, 0);
    delay(100);
}
```

### Цикл while

Еще один тип циклов – цикл **`while`**. Его используют, если нужное количество действий **заранее неизвестно** или **неважно**. Он может выполниться несколько раз, один раз, бесконечное количество раз — или вообще не выполниться ни разу.\
Слово while переводится как «пока» и отражает суть: **пока верно какое-то условие, цикл выполняется.**

Примеры использования цикла while в жизни:

* Вы ждете на остановке, пока не приедет автобус
* Пока светло, робот Валли убирает мусор.
* Пока температура больше 25 градусов, работает кондиционер.
* Пока не нажата кнопка, будильник издает звуковой сигнал

Структура цикла while:

```arduino
while (условие) {
    инструкция;
}
```

По сути while работает как if, но после выполнения инструкции возвращается обратно к проверке условия.

{% hint style="info" %}
Часто if и while в ардуино могут работать одинаково, потому что основной код программы мы пишем в функции loop(), то есть он бесконечно повторяется, а значит программа из раза в раз будет возвращаться к проверке условия в if. Тем не менее, цикл while тоже часто может быть нужен!
{% endhint %}

Пример: лампа горит, пока кнопка нажата (здесь как раз нет разницы с if, если писать этот код в loop):

```arduino
while (knopka_state == 1) {
    digitalWrite(lampa, 1);
}
```

### Цикл do..while

### Цикл do while <a href="#id-2-toc-title" id="id-2-toc-title"></a>

`do` - **цикл с постусловием**, то есть сначала выполняется инструкция, а затем проверяется условие. Дословно - _делать, пока_ `условие` верно. Синтаксис такой:

```cpp
do {
    инструкция;
} while (условие);
```

Таким образом инструкция или блок кода будут выполнены **как минимум один раз**, и только потом проверится условие.

### Управление циклами: break и continue

**Оператор break** ("_прервать"_) используется для принудительного выхода из циклов do, for или while по какому-нибудь условию не дожидаясь завершения цикла по условию самого цикла:

```arduino
int i = 5;
while (i > 0) {
    i -= 1; // то же самое, что i = i-1 или i--
    if (i == 2) break;
    // здесь i примет значения 4, 3, а затем цикл прервется
}
```

Пример - покидаем цикл, если кнопка нажата:

```arduino
while (true) {
    // код...
    if (кнопка_нажата) break;
}
```

**Оператор continue** ("_продолжить_")  пропускает оставшиеся операторы в текущем шаге цикла и программа перепрыгивает на проверку условия цикла

```arduino
int i = 5;
while (i > 0) {
    i -= 1;
    if (i > 2) continue;
    // здесь i примет значения 2, 1, 0
}
```
