# Проект: Светофор

### Задание

Разработать систему управления светофором.

Предоставьте блок-схему алгоритма, соберите устройства, напишите код, разработайте и заполните тест-кейсы.

**Компоненты на выбор:**

* Вариант 1 (Простой): Arduino Uno, макетная плата, кнопка, 3 светодиода (красный, желтый, зеленый), 3 резистора \~220 Ом, 1 резистор \~10 кОм, провода.
* Вариант 2 (Компактный): Arduino Uno, макетная плата, кнопка, 1 RGB-светодиод, 3 резистора \~220 Ом, 1 резистор \~10 кОм, провода.

**Описание алгоритма:**

Исходное состояние: Светофор выключен.

Запуск: Первое нажатие на кнопку запускает рабочий цикл светофора.

Рабочий цикл (длительность 14 секунд):

* Красный (5 секунд): Горит постоянно.
* Желтый (2 секунды): Горит постоянно.
* Зеленый (3 секунды): Горит постоянно.
* Зеленый мигающий (2 секунды): Мигает с частотой 2 Гц (включен 0.5 сек, выключен 0.5 сек, и так 4 раза).
* Желтый (2 секунды): Горит постоянно.

Цикл повторяется 1 раз, далее ожидание нажатия на кнопку.

_**Во время работы цикла нажатия на кнопку игнорируются.**_

<summary>Подсказки по программе</summary>

Введите дополнительную переменную-флаг типа int, например можно назвать ее svetofor\_rabotaet. Она будет равна 0 или 1 в зависимости от того, включен сейчас основной цикл светофора или нет. Это поможет выполнить условие "Во время работы цикла нажатия на кнопку игнорируются.", то есть, если сейчас  svetofor\_rabotaet == 1, то ничего запускать не будем. А если  svetofor\_rabotaet == 0, то значит светофор не работает и можно его запустить

**Основной цикл (loop):**

1. Считать состояние кнопки.
2. Если кнопка нажата и светофор неактивен (флаг svetofor\_rabotaet == 0)?

_**ЕСЛИ ДА:**_

3. Установить флаг svetofor\_rabotaet = 1 (начать работу, заблокировать кнопку).
4. Цикл "Светофор":
   1. Включить Красный. Задержка 5000 мс.
   2. Выключить Красный. Включить Желтый. Задержка 2000 мс.
   3. Выключить Желтый. Включить Зеленый. Задержка 3000 мс.
   4. Мигание Зеленым 4 раза (2Гц = по 0.5 сек вкл\выкл):
   5. Выключить Зеленый. Включить Желтый. Задержка 2000 мс.
   6. Выключить Желтый.

5\. Сбросить флаг svetofor\_rabotaet = 0 (разблокировать кнопку для следующего запуска).

_**ЕСЛИ НЕТ:**_

6. Продолжить ожидание.

Примерно так это может выглядеть в коде:

```arduino
if (knopka_state == 1 && svetofor_rabotaet == 0) { 
    svetofor_rabotaet = 1; // Устанавливаем флаг активности
    
    //здесь весь алгоритм лампочек

    svetofor_rabotaet = 0;
}
```
