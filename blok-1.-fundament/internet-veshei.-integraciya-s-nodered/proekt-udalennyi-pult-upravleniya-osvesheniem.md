# Проект: удаленный пульт управления освещением

* **Задача:** Создать дашборд в Node-RED с несколькими кнопками. При нажатии на кнопку в Node-RED на плате Arduino загорается соответствующий светодиод.
* **Практика:** Узлы ui\_button, function (для формирования команды, например, строки "LED1\_ON"), serial out. В Arduino — код, который читает данные из COM-порта и парсит команды.
* **Документация:**
  1. **Архитектурная схема:** Как на рис. 1 в вашем файле, но адаптированная. \[Пользователь] -> \[Dashboard] -> \[Node-RED] -> \[Serial (USB)] -> \[Arduino].
  2. **Таблица параметров управления:** Описание команд, которые отправляются в Arduino (например, "LED1\_ON", "LED1\_OFF").
